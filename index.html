About MiMiC:

MiMiC is a single-player, memory game inspired by the traditional 1978 Hasbro game - Simon. MiMiC the memory game shares similar rules and also consists of four coloured buzzers, which each light up and play a sound when pressed. Game Instructions - How to play:

Mimic® is a single-player, memory game with 30 levels. Your aim is to correctly repeat/mimic the buzzers that the system selects in order to move on to the next level.

  1. When you press the start button, the game's starting theme song will play. After this, the game will make the first move.
  2. For Level 1, the game will select one buzzer at random, the selected buzzer's light will flash and play a sound.
  3. After this, you now have to reselect that buzzer selected by the game.
  4. When you have correctly repeated a sequence, a high-pitched notification sound will be played before the next level begins. If a wrong buzzer has been pressed, then the game will release a low-pitched notification sound and the game will end.
  5. For each new level, one extra buzzer is selected and added to the sequence. It could be the same buzzer as the previous selection or a different one so watch carefully.
  6. You will have to replay all of the buzzers that have been selected so far in their exact order, including the newly selected buzzer for that level. For example on level 4, the user will have to replay the three buzzers selected so far, in their order of selection and the fourth buzzer selected in that level. The game can continue up to a maximum of 30 levels.

Repeat the pattern of lights and sounds for as long as you can! Best of Luck!
