Game Instructions - How to play:

Mimic® is a single player memory game inspired by the traditional Hasbro game - Simon. Mimic® has 30 levels and the system begins the game by selecting one buzzer. The user's aim is to correctly repeat(mimic) the buzzers that the system selects in order to move on to the next level. For each new level, one extra buzzer is selected and added to the sequence. It could be the same button as the previous selection or a different button. The user will now have to replay all of the buzzers that have been selected so far in their exact order, including the newly selected buzzer for that level. (For example on level 4, the user will have to replay the three buzzers selected so far, in their order of selection and the fourth buzzer selected in that level).

When a sequence has been correctly repeated by the user, a high-pitched notification sound will be played before the next level begins. If a wrong buzzer has been pressed, then the game will release a low-pitched notification sound and the game will end.

  • Start Button - Begins a new game
  • Hint Button - Replays the last buzzer selected in the sequence so far
